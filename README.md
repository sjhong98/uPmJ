<br/><br/>

'넌 P해 난 J해'(이하 UPMJ)는 MBTI가 J인 계획형 인간들이 여행 계획을 쉽고 편하게 세울 수 있도록 만들어진 서비스입니다. <br/>
공공데이터를 이용하여 수많은 여행지를 블록쌓듯 계획을 세우고, 지도 상에 여러분이 세운 여행 동선을 직접 확인할 수 있습니다! <br/>
물론, 여행은 친구들과 함께해야 제맛이겠죠, 여러분의 친구들과 동시에 계획을 세워보세요!
